# NumPy and Pandas Assignment

## Overview

This project demonstrates basic data analysis tasks using **Python**, **NumPy**, and **Pandas**.
The objective is to practice common operations used in data analysis such as array manipulation, statistical calculations, data filtering, sorting, and file handling.

## Project Workflow

The Python script performs the following steps:

1. Creates NumPy arrays and matrices.
2. Performs mathematical operations such as addition, multiplication, and dot product.
3. Generates random matrices.
4. Reshapes arrays into different dimensions.
5. Creates a Pandas DataFrame for employee data.
6. Performs data analysis operations such as:

   * Viewing the first rows
   * Calculating mean salary
   * Filtering employees based on salary
   * Selecting specific columns
   * Adding a Bonus column
   * Sorting employees by salary
7. Displays data types and summary statistics.
8. Saves the dataset as a CSV file.
9. Loads the CSV file and checks the dataset shape.

## Files in this Repository

* `numpy_pandas_assignment.py` – Python script containing all solutions.
* `employee_data.csv` – Dataset generated during the assignment.
* `README.md` – Project documentation.

## Tools Used

* Python

## Purpose

The purpose of this assignment is to build a strong foundation in **data manipulation and analysis**, which are essential skills for data analysts and data scientists.
